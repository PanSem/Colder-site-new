All the colors under your cursor

Find every pixel's color value in your Computer. Open the program and move the cursor around. You will see that it finds every color in your screen.

Download Now

What can you do?

Color Value

Move the mouse in the pixel you want and its color value will be immediately showed up.

Copy Hex

You can copy every pixel's hex color value you want and use afterwards everywhere you want.

Export File

You can save in a txt file every important color value in rgb and hex format and name it.
